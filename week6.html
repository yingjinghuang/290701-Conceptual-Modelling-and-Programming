Week 6: Midterm Exam

18.05.2026 · 2 hours · Paper-based

Format

The exam is paper-based and runs for two hours. It covers Weeks 1–5. The booklet has three parts, totalling 100 points:

  • Part A — Multiple Choice. 25 questions, 2 points each (50 pts). Mark exactly one box per question on the answer form.
  • Part B — Read Code and Predict Output. 2 questions, 10 points each (20 pts). Write the program output on the answer form.
  • Part C — Write Code. 2 questions, 15 points each (30 pts). Write your Java source code on the answer form.

You will write source code on paper. Minor syntax slips (a missing semicolon, a forgotten import) will not cost full marks — what we grade is whether the code expresses the right idea.
